# 😴 SnoozeGuard
### Умный таймер сна для ПК с защитой от зависаний.

## 🤔 Что это за программа 
Данная программа занимается отслеживанием взаимодействия пользователя с компьюетром, собирает данные процессов и на анализе всех этих данных решает, отправлять ли ваш компьютер в режим сна.
